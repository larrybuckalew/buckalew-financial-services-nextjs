| Version | Supported          |
| ------- | ------------------ |

## Reporting a Vulnerability

### Responsible Disclosure
If you discover a security vulnerability in the Buckalew Financial Services project, we appreciate your help in disclosing it to us in a responsible manner.

#### Process
1. Email security details to: security@buckalew-financial.com
2. Provide a detailed description of the vulnerability
3. Include steps to reproduce
4. Do not publicly disclose the issue before we've had a chance to address it

### What to Expect
- We aim to respond within 48 hours
- We will keep you informed about the progress
- Credit will be given to responsible reporters

## Security Best Practices
- Always use the latest version
- Keep dependencies updated
- Use strong, unique passwords
- Enable two-factor authentication
- Report any suspicious activities

## Our Commitment
- We take security seriously
- We respond quickly to potential vulnerabilities
- We maintain transparent communication
